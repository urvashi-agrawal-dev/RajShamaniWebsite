# Raj Shamani - Premium Personal Brand Website

A cinematic, image-first personal brand website for Raj Shamani - Podcaster, Entrepreneur, and Author.

## 🎯 About

This is a **concept website** created for educational and portfolio purposes. It is **NOT** an official website.

## ✨ Features

- **Full-Screen Hero Section** with cinematic animations
- **Authority Section** showcasing conversations with global icons (Bill Gates, Vijay Mallya, Michael Phelps, Sunita Williams)
- **Podcast Section** featuring "Figuring Out with Raj Shamani"
- **Impact & Stats** with animated counters (14.7M+ subscribers, 1.7B+ views)
- **Story & Journey** highlighting key milestones
- **Book Section** for "Build, Don't Talk"
- **Media & Recognition** featuring Forbes and other achievements
- **Collaborations & Speaking** opportunities
- **Social Connect** section with all platforms

## 🎨 Design System

- **Colors**: Yellow (#FFD700), White, Black
- **Typography**: Space Grotesk (headings), Inter (body)
- **Style**: Premium, cinematic, image-first, minimal text
- **Animations**: GSAP-powered smooth animations

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
- GSAP (GreenSock Animation Platform)
- Fully Responsive Design
